# 合规复习题库网站

这是一个本地运行的复习题库原型，适合整理老师允许分享的练习题、公开往年题、自己整理的错题和考试结束后允许复盘的材料。

## 怎么打开

直接双击 `index.html`，或在浏览器里打开这个文件。

## 已有功能

- 上传题目图片并进行 OCR 文字识别
- 手动编辑识别结果
- 保存课程、章节、题型、难度、标签、答案
- 本地题库列表、搜索和筛选
- 相似题/重复题检测
- 随机刷题模式
- 生成解析草稿和可复制的 AI 提示词
- 导出题库 JSON

## 数据保存在哪里

题目会保存在当前浏览器的 `localStorage` 里，不会自动上传到服务器。换浏览器或清理浏览器数据后，本地题库可能会消失，所以重要内容建议定期导出 JSON。

## 下一步可升级

- 用后端数据库保存题库
- 接 OpenAI API 自动生成更完整的解析
